Errors
Compile-time errors:

Errors found by the compiler.

There are two types of compile-time errors:
  1. Syntax errors**: Errors that occur when we violate the rules of C++ syntax.
    2. Some common syntax errors are:
    • Missing semicolon `;`
    • Missing closing parenthesis `)`, square bracket `]`, or curly brace `}`
  2. Type errors**: Errors that occur when there are mismatch between the types we declared.
    3. Some common type errors are: - Forgetting to declare a variable - Storing a value into the wrong type
